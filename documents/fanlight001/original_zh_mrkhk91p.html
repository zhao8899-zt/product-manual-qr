风扇灯安规认证信息

产品型号：FL-2024-Pro

一、产品认证信息

1. CCC认证：中国强制性产品认证（证书编号：2024010701001234）

2. CE认证：欧盟符合性认证（证书编号：CE-FL2024-EU-0567）

3. FCC认证：美国联邦通信委员会认证（FCC ID：2A3FL-FL2024P）

4. PSE认证：日本电气用品安全法认证（证书编号：PSE-JP-FL-2024-089）

5. RoHS认证：有害物质限制指令合规

6. 能效标识：中国能效标识1级（最节能等级）

二、电气安全参数

1. 额定电压：AC 220V 50Hz（中国版）/ AC 110V 60Hz（北美版）/ AC 230V 50Hz（欧洲版）

2. 防护等级：IP20（室内使用，不防水）

3. 绝缘等级：Class II（双重绝缘）

4. 接地方式：带接地保护

5. 漏电流：<=0.25mA（符合GB 4706.1标准）

6. 耐压测试：1500V/1分钟无击穿

7. 温升测试：电机绕组温升<=75K，外壳温升<=45K

三、安全使用警示

警告：本产品仅适用于室内干燥环境，请勿在浴室、游泳池等潮湿场所使用。

警告：请勿在产品运行时触碰扇叶，以免受伤。

警告：请勿用湿手插拔电源插头或操作开关。

警告：产品附近请勿放置易燃易爆物品。

警告：请勿自行拆卸产品内部结构，触电危险！

注意：长时间不使用时，请拔掉电源插头。

注意：请使用产品原装电源线，请勿使用延长线或多功能插座。

四、电磁兼容性（EMC）

1. 辐射骚扰：符合GB 4824/CISPR 11 Class B标准。

2. 传导骚扰：符合GB 4824/CISPR 11 Class B标准。

3. 谐波电流：符合GB 17625.1标准。

4. 电压波动和闪烁：符合GB 17625.2标准。

5. 静电放电抗扰度：通过IEC 61000-4-2测试，4级（8kV接触放电）。

6. 射频电磁场辐射抗扰度：通过IEC 61000-4-3测试，3级（10V/m）。

五、材料环保信息

1. 产品外壳材质：阻燃ABS（UL94 V-0级）。

2. 扇叶材质：AS+玻纤增强，通过RoHS检测。

3. 不含铅、汞、镉、六价铬、多溴联苯（PBB）和多溴二苯醚（PBDE）。

4. 包装材料：可回收纸浆模塑+瓦楞纸箱，环保可降解。

5. 产品中有害物质含量符合GB/T 26572-2011标准。

六、能效与节能

1. 能效等级：1级（GB 12021.9标准最高等级）。

2. 待机功耗：<=0.5W。

3. 功率因数：>=0.9。

4. 相比传统交流电机风扇，本产品节能40%以上。

5. LED光源光效：>=100lm/W。

七、废弃处理

1. 本产品含有电子元器件，请勿与生活垃圾一同丢弃。

2. 请将废弃产品送至指定电子废弃物回收点。

3. 遵循废弃电器电子产品回收处理管理条例相关规定。

4. 正确回收有助于保护环境和人体健康。
