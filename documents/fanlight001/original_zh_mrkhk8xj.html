风扇灯使用说明书

产品型号：FL-2024-Pro

一、产品概述

FL-2024-Pro智能风扇灯是一款集照明、风扇、氛围灯于一体的智能家居产品。采用无刷直流电机，低噪音、大风量、节能环保。支持遥控器控制和手机APP控制，为您打造舒适的生活空间。

二、产品参数

额定电压：AC 220V 50Hz

额定功率：风扇36W + 照明24W = 60W

风扇转速：3档可调（低速/中速/高速）

风量：最大3500m3/h

噪音等级：<=35dB（低速）

照明色温：3000K-6500K可调（暖光/自然光/白光）

LED寿命：>=50000小时

产品尺寸：直径560mm x 高180mm

净重：3.5kg

三、遥控器功能说明

1. 电源键：短按开/关风扇，长按3秒开/关照明。

2. 风速键：循环切换低/中/高三个档位。

3. 模式键：自然风/睡眠风/强力风三种模式。

4. 灯光键：暖光/自然光/白光循环切换。

5. 亮度键：照明亮度无极调节（按住渐变）。

6. 定时键：1小时/2小时/4小时/8小时定时关闭。

7. 反转键：风扇正反转切换（夏天正转降温，冬天反转循环暖气）。

8. 配对键：与风扇灯进行遥控配对。

四、APP智能控制

1. 扫描包装盒上的二维码下载智能家APP。

2. 注册账号并登录。

3. 确保手机连接2.4GHz WiFi网络。

4. 打开APP，点击添加设备，选择风扇灯。

5. 按提示长按遥控器配对键3秒，等待设备连接。

6. 连接成功后可远程控制、设置定时、语音控制（支持小爱同学、天猫精灵、小度）。

五、日常使用建议

1. 夏季使用：选择正转+高速档，配合空调可节能30%。

2. 冬季使用：选择反转+低速档，将暖气向下循环，提升取暖效果。

3. 睡眠模式：自动降速并定时关闭，营造舒适睡眠环境。

4. 照明建议：阅读时使用自然光（4000K），休息时使用暖光（3000K）。

六、清洁保养

1. 清洁前请先关闭电源。

2. 用柔软干布擦拭灯罩和扇叶，请勿使用腐蚀性清洁剂。

3. 每月清理一次扇叶灰尘，保持最佳风量。

4. 每年检查一次安装螺丝是否松动。
